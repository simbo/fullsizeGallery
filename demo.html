fullsizeGallery

… is a jQuery plugin by Simon Lepel for displaying a fullsize image or gallery in any HTML container.

Demos

  • Simple Fullscreen single image using body as container
  • Fullscreen Slideshow slideshow with custom effects and optimized performance
  • Fullscreen Gallery slideshow with multiple controls
  • Two Slideshows on Splitscreen two independent slideshows in separated div containers
